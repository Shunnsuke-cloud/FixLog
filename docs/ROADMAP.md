## 各フェーズの開始方法

### フェーズを開始する際のチェックリスト

```bash
# 1. develop ブランチを最新に
git checkout develop
git pull origin develop

# 2. フェーズ用ブランチを作成
git checkout -b feature/phase-N-機能名

# 3. 実装

# 4. テスト・確認
npm run dev
npm run type-check
npm run lint

# 5. コミット
git add .
git commit -m "feat: フェーズN 機能説明"

# 6. プッシュ・PR 作成
git push origin feature/phase-N-機能名
```
